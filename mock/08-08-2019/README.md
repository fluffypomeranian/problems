### run time O()
The time complexity is O(n), n is the length of the input.
The space complexity is O(n), n is the size of the input.


###problem statement
```
Given a string, zero pad any whole number to X char.

Examples:
"area 59", 4
return "area 0059"
"123 Foo st", 5
return "00123 Foo st"
"123 Foo st", 2
return "123 Foo st"
"Area59asdf234", 4
return "Area0059asdf0234"
```
